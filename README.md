## 🌍 Track Your Impact. Save the Planet.

**EcoTrack** is a powerful, user-friendly sustainability dashboard designed to help you understand and reduce your carbon footprint. By logging your daily habits—from commuting to meals—you can visualize your environmental impact and receive actionable, data-driven recommendations to live a greener life.

### ✨ Key Features

- **📊 Real-time Tracking**: Log activities across categories like Transport, Energy, and Diet.
- **📈 Impact Visualization**: See your carbon emissions broken down by category with interactive charts.
- **💡 Smart Recommendations**: Get personalized tips based on your highest emission areas.
- **🏆 Gamified Progress**: Earn a "Green Score" and track your journey towards a zero-waste lifestyle.
- **🔒 Secure & Private**: Your data is stored securely using Firebase.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- A Firebase project (for authentication and database)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aajvr/EcoTrack.git
   cd EcoTrack
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your Firebase configuration (see `.env.example` for the template):
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   # ... add other variables
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
